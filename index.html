Docxtend

Xtend annotations to create object builders

Eclipse update site

Xtend active annotations which helps to create object builders

org.mv.docxtend is the main plugin containing the annotation definition ... in fact the file XtendTree.xtend is the only one you need to create builders.

org.mv.docxtend.tools contains tools and eclipse documentation.

org.mv.docxtend.eclipse.... are lazy builders for eclipse

org.mv.alpha is a sample using the eclipse builders
