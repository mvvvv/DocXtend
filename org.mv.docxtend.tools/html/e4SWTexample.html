The E4/SWT example

Introduction


The project org.mv.alpha contains a RCP Product example that is at first, an Xtend reproduction of the custom samples provided from the New Eclipse 4 Application Project wizard then some dummy parts have been added for demonstration

Installation


This project has been created with 3 builders:
  • E4 builder from plugin "org.mv.docxtend.eclipse.e4"
  • SWT builder from plugin "org.mv.docxtend.eclipse.swt"
  • JFace builder from plugin "org.mv.docxtend.eclipse.jface"
these plugins are installed with DocXtend tools and annotations library.

To run this example you have to install (thanks to vogella.com and Lars Vogel):
  • Eclipse Kepler 4.3
  • Eclipse 4 tooling for Kepler
Download Xtext/Xtend (v2.4.2 minimal) from your main central site (Modeling category). Check also Xtend site

Description


In the plugin.xml file, The extension "org.eclipse.core.runtime.products" has be enhanced with a new lifeCycleURI property indicating where to find the E4Application definition: org.mv.alpha.E4Application.xtend



The file E4Application.e4xmi is made of an empty <application:Application .../> element

All the Menu Handlers share the same compilation unit org.mv.alpha.handlers.AllMenuHandlers.xtend

All the contribution parts share the same compilation unit org.mv.alpha.parts.AllParts.xtend

Launch the example


Open the file org.mv.alpha.product then click on "overview tab »launch an eclipse application"
